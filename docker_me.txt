1. create docker image
    1.1 command
        docker build -t willsofts/mext .
    1.2 check image
        docker images

2. test run docker image
    2.1 run
        docker run -d --name mext --env VALIDATE_TOKEN=false -p 8080:8080 willsofts/mext
        docker run -d --name mext -p 8080:8080 willsofts/mext
    2.2 try to simulate test by calling api:
        curl http://localhost:8080/api/config/list
    2.3 stop & remove container
        docker stop mext
        docker rm mext

3. push image to docker hub
    3.1 login
        docker login
        docker login --username willsofts --password Tassan@2024
    3.2 tag
        docker tag willsofts/mext willsofts/mext:1.0.0
    3.3 tag latest
        docker tag willsofts/mext willsofts/mext:latest
    3.4 push
        docker push willsofts/mext:1.0.0
        docker push willsofts/mext:latest
    remark: can use docker_push.bat
    goto: https://hub.docker.com search willsofts
    ex. https://hub.docker.com/r/willsofts/mext
    caution: try to check latest version before tag and push

4. pull image
    4.1 pull
        docker pull willsofts/mext:latest
    4.2 run test
        docker run --name mext --env VALIDATE_TOKEN=false -p 8080:8080 willsofts/mext
    remark: can use docker-compose.yml
